Barra Listrada

Adicionar .progress-bar-striped a qualquer .progress-bar para aplicar uma faixa via gradiente CSS sobre a cor de fundo da barra de progresso.




Barra Listrada animada

O gradiente listrado também pode ser animado. Adicione .progress-bar-animated a .progress-bar para animar as listras da direita para a esquerda por meio de animações CSS3.
